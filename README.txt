====================================
ГАЙД ПО ИСПОЛЬЗОВАНИЮ MiSide MOD MANAGER
====================================

1. СТРУКТУРА ПАПКИ
==================
После распаковки у тебя должна быть такая структура:

InstallerMisideMods/
├── MiSideModInstaller.exe     - главный файл (установщик)
├── mods/                       - сюда кидаешь моды
├── backups/                     - папка с бэкапами
├── logs/                        - логи установок
└── temp/                         - временные файлы

2. КАК ДОБАВИТЬ МОДЫ
====================
1. Открой папку "mods"
2. Скопируй туда моды/скины:
   - Можно кидать .pak файлы
   - Можно кидать .zip архивы
   - Можно кидать целые папки
   - Можно кидать картинки .png .jpg (для скинов)

3. КАК УСТАНАВЛИВАТЬ
====================
1. Запусти MiSideModInstaller.exe
2. Нажми "Обзор" и укажи путь к игре MiSide
   (обычно: C:\Program Files (x86)\Steam\steamapps\common\MiSide)
3. В левом окне выбери моды которые хочешь установить
4. Нажми кнопку "Установить выбранные"
5. Дождись сообщения об успешной установке

4. ГДЕ ИСКАТЬ МОДЫ ПОСЛЕ УСТАНОВКИ
==================================
Все моды устанавливаются в папку игры:
MiSide\Data\Custom\

5. БЭКАПЫ
=========
Перед установкой программа автоматически создает бэкап
в папке "backups". Если что-то пойдет не так - файлы оттуда
можно скопировать обратно в игру.

6. ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
==========================
- Проверь что путь к игре указан правильно
- Закрой игру перед установкой модов
- Посмотри логи в папке "logs" - там видно ошибки
- Убедись что мод подходит для твоей версии игры

7. ПОЛЕЗНОЕ
===========
- Можно устанавливать несколько модов сразу
- Программа сама распаковывает ZIP архивы
- Все операции записываются в лог
- Папку "temp" можно иногда чистить (временные файлы)

====================================
Все просто! Кидаешь моды в mods,
выбираешь в программе и устанавливаешь!
