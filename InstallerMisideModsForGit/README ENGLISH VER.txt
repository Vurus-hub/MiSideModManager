=======================================
MiSide MOD MANAGER USER GUIDE
=========================================

1. FOLDER STRUCTURE
=====================
After unpacking, you should have the following structure:

InstallerMisideMods/
├── MiSideModInstaller.exe - main file (installer)
├── mods/ - this is where you put mods
├── backups/ - folder With backups
├── logs/ - installation logs
└── temp/ - temporary files

2. HOW TO ADD MODS
======================
1. Open the "mods" folder
2. Copy mods/skins there:
- You can upload .pak files
- You can upload .zip archives
- You can upload entire folders
- You can upload .png or .jpg images (for skins)

3. HOW TO INSTALL
=======================
1. Run MiSideModInstaller.exe
2. Click "Browse" and specify the path to the MiSide game
(usually: C:\Program Files) (x86)\Steam\steamapps\common\MiSide)
3. In the left window, select the mods you want to install.
4. Click "Install Selected"
5. Wait for the successful installation message.

4. WHERE TO FIND MODS AFTER INSTALLATION
=====================================
All mods are installed in the game folder:
MiSide\Data\Custom\

5. BACKUPS
==========
Before installation, the program automatically creates a backup
in the "backups" folder. If something goes wrong, the files from there
can be copied back into the game.

6. IF SOMETHING DOESN'T WORK
=============================
- Check that the game path is correct
- Close the game before installing mods
- Check the logs in the "logs" folder - errors are visible there
- Make sure the mod is compatible with your game version

7. USEFUL
=============
- You can install multiple mods at once
- The program automatically unpacks ZIP archives
- All operations are recorded in the log
- You can occasionally clean the "temp" folder (temporary files)

========================================
It's simple! Just drop mods into the "mods" folder,
select them in the program, and install!
